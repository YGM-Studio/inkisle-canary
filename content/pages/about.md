---
title: "关于本站"
summary: "这是一个从 npm 正式版本构建的 InkIsle 外部验收站。"
---

这个站点由 YGM Studio 维护，用来持续验证 npm 上公开发布的 InkIsle 版本。

仓库只保存 Markdown、配置和部署工作流。渲染器、主题与 Astro 实现全部来自 npm 包，因此它不会因为本地源码恰好可用而掩盖发布包问题。

- [InkIsle 源码](https://github.com/YGM-Studio/inkisle)
- [InkIsle npm 包](https://www.npmjs.com/package/inkisle)
