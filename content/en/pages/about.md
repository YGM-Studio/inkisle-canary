title: "About"
summary: "An external InkIsle acceptance site built from the published npm release."
---

YGM Studio maintains this site to continuously validate public InkIsle releases from npm.

The repository contains only Markdown, configuration, and deployment workflows. Its renderer, themes, and Astro implementation all come from the npm package, preventing a local source checkout from hiding packaging defects.

- [InkIsle source](https://github.com/YGM-Studio/inkisle)
- [InkIsle on npm](https://www.npmjs.com/package/inkisle)
